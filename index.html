Synopsis

Hi, my name is Clayton, and I'm documenting my journey towards learning the C++ language.  This website is a repository of subjectively interesting C++ topics, books, and references.  My end-goal is to enable myself (and help others along the way) to effectively harness the performance benefits of C++.
